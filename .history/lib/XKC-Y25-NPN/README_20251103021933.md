# XKC-Y25-NPN Bibliothek

Diese Bibliothek kapselt die Nutzung des berührungslosen Füllstandssensors XKC-Y25-NPN (NPN, digital).

## Kurzbeschreibung

- Versorgung: 5–12 V (Sensor-Platine)  
- Ausgang: NPN open-collector (zieht auf GND, wenn Wasser erkannt)  
- Für den Anschluss an einen 3.3V-Mikrocontroller ist ein Pull‑Up auf 3.3V empfehlenswert (R ≈ 10 kΩ).  
(Alle diese Hinweise basieren auf Projektdokumentation und den Messhinweisen).

## Installation

- Leg das Verzeichnis `XKC_Y25_NPN` in den `lib/`-Ordner deines PlatformIO-Projekts.
