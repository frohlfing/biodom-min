# GY302_BH1750

Diese Bibliothek kapselt die Nutzung des Lichtsensors GY-302 (BH1750) in einer kompakten, leicht testbaren Klasse.

## Eigenschaften

- Direkte I2C-Ansteuerung ohne externe BH1750-Bibliothek
- Messung in Lux (float)
- Einfache Fehlercodes und Fehlermeldungen
- Beispielsketch für serielle Ausgabe kompatibel mit Serial Plotter

## Installation

Kopiere das Verzeichnis `GY302_BH1750` in den `lib/`-Ordner deines PlatformIO-Projekts oder füge die Dateien in dein Projekt ein.

Keine zusätzlichen Bibliotheken erforderlich — die Klasse verwendet die Standard-`Wire`-Bibliothek.
